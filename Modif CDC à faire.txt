PAGE BIOGRAPHIE :
La page biographie s'ouvrira avec une jaquette de jeu en 3D avec comme image mon logo. Une animation se lancera afin d'ouvrir la jaquette qui dévoilera le CD du jeu "Matsashi, The Game" ainsi qu'un livret sur la gauche affichant un sommaire de la présentation, des projets et des partenaires de jeu récurrents.
Les articles et le contenu se trouveront en dessous.

PAGE BIBLIOTHÈQUE :
-Ajouter toutes les consoles
-Ajouter tous les constructeurs
-Ajouter les filtres pour trier

PANEL ADMIN : 
Accueil => le panel des options avec :
- Ajouter un jeu
- Ajouter un support
- Modifier un jeu (modifier les infos, les vidéos/clips, temps & taux et supprimer le jeu)
- Supprimer un support

Modifier un jeu => L'admin arrive sur une page avec une barre de recherche (datalist) et en dessous la liste de tous les jeux (max 50 jeux par page, rendu comme le TP POOBibli).
Chaque jeu affichera la jaquette, le nom du jeu, le support et possédera deux boutons, un pour modifier le jeu et un pour le supprimer.

La modification affichera toutes les informations déjà renseignées pour le jeu dans un formulaire et la modification d'un champ écrasera la donnée changée dans la BDD.

La suppression fera apparaître une pop-up demandant de confirmer, puis une fois confirmé le jeu sera supprimé de la BDD.

Ajouter un jeu => Formulaire avec jaquette en plus

Ajouter un support => Formulaire avec logo de la console

Supprimer un support => Tableau avec les supports actuelles (comme sur le TP POOBibli) comprenant :
- Logo du support
- Nom du support
- Nom du constructeur
- Un bouton de suppression
